
## Troubleshooting

### Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| `AADSTS700016: Application not found` | Wrong tenant ID or app not in your tenant | Verify tenant ID, ensure app is in correct directory |
| `AADSTS7000215: Invalid client secret` | Secret expired or incorrect | Create new secret, copy value correctly |
| `AADSTS65001: User or admin has not consented` | Admin consent not granted | Ask IT admin to grant consent |
| `AADSTS700027: Client assertion contains invalid signature` | Wrong secret used | Ensure you're using the "Value", not "Secret ID" |
| `403 Forbidden` on Graph API | Permissions granted but not for this operation | Verify correct permissions are consented |

### Finding Your AAD Object ID

To find your own Azure AD Object ID for testing:

1. Go to Azure Portal â†’ Azure Active Directory
2. Click "Users" in the left menu
3. Search for your name
4. Click your user entry
5. Copy the "Object ID" field

### Secret Rotation

Secrets expire after 24 months. Set a calendar reminder to:

1. Create new secret before expiration
2. Update your bot configuration with new secret
3. Delete old secret after verifying new one works
